## Python學習資源彙總

有很多小夥伴都在網上找 Python 相關的學習資源，但是網上的資源良莠不齊，有些所謂的高贊內容其實並不能幫到真正有學習需求的人。這裡，我幫大家把優秀的學習資源做一個彙總，讓大家少走一些彎路，希望這個內容對大家有所幫助。

> **溫馨提示**：下面羅列的資源選適合自己的就行了，並非每個都值得你去學習和研究。

#### 資源網站

10. [Kaggle](https://www.kaggle.com/) 官方網站 - 全球知名的資料科學和機器學習平臺
11. [GitHub](https://github.com/) 官方網站 - 全球最大程式碼託管平臺，上面有很多優質的程式碼和資源

13. [Bilibili](https://www.bilibili.com/) 網站 - 原本是一個二次元平臺，現在也是年輕人的學習平臺
14. [中國大學 MOOC](https://www.icourse163.org/)、[網易雲課堂](https://study.163.com/)、[慕課網](https://www.imooc.com/)、[Udemy]()、[Coursera]()、[Udacity]()、[edX](https://www.edx.org/) - 線上學習平臺

16. [力扣](https://leetcode.cn/)、[牛客網](https://www.nowcoder.com/)、[HackerRank](https://www.hackerrank.com/)、[topcoder](https://www.topcoder.com/) - 線上刷題、比賽平臺

### Python參考書籍

#### 入門讀物

1. 《Python程式設計從入門到實踐》（*Python Crash Course*）- 著名的蟒蛇書，推薦
2. 《Python基礎教程》（*Beginning Python From Novice to Professional*）- 入門經典
3. 《Python學習手冊》（*Learning Python*）- 著名的老鼠書
4. 《Python程式設計導論》（*Introduction to computation and programming using Python*）- 我自己很喜歡這本書

#### 進階讀物

1. 《Python Cookbook》- 照著菜譜做，肯定不出錯，進階首選
2. 《流暢的Python》（*Fluent Python*）- 成為高手的必經之路
3. 《Effective Python：編寫高質量Python程式碼的59個有效方法》（*Effective Python 59 Specific Ways to Write Better Python*）
4. 《Python高階程式設計》（*Expert Python Programming*）- 著名的花書
5. 《Python專案開發實戰》- 有很多工程化專案開發的知識

#### 資料採集

1. 《Python 3網路爬蟲開發實戰》- 個人推薦
2. 《Python網路書籍擦劑》- 著名的穿山甲書

#### 資料分析

1. 《利用Python進行資料分析》（*Python for Data Analysis*）- pandas 作者寫的書
2. 《Python資料科學手冊》（*Python Data Science Handbook*）- O'Reilly 動物數系列的鬣蜥書
3. 《Python資料分析實戰》（*Python Data Analytics With Pandas, NumPy, and Matplotlib*）- 個人推薦
4. 《Python資料視覺化程式設計實戰》（*Python Data Visualization Cookbook*）- 重點在資料視覺化
5. 《深入淺出資料科學》（*Principles of Data Science*）- 我自己很喜歡這本書
6. 《面向資料科學家的實用統計學》（*Practical Statistics for Data Scientists*）- 很實用的統計學
7. 《資料科學入門》（*Data Science from Scratch*）

#### 機器學習

1. 《機器學習實戰》（*Machine Learning in Aciton*）- 入門推薦
2. 《機器學習（鳶尾花數學大系：從加減乘除到機器學習）》- 姜偉生博士非常用心的作品
3. 《深度學習入門：基於Python的理論與實現》- 日本獲獎書籍，入門推薦
4. 《深度學習》（*Deep Learning*）- 深度學習經典神作
5. 《動手學深度學習》（*Dive into Deep Learning*）- 阿斯頓·張和李沐強強聯合出品
6. 《Python深度學習》（*Deep Learning with Python*）- 我自己很喜歡這本書
7. 《機器學習實戰》（*Hands-On Machine Learning with Scikit-Learn, Keras, and Tensorflow*）- 著名的蜥蜴書，從機器學習到深度學習

#### 大模型
