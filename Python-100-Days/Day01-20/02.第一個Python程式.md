## 第一個Python程式

在上一課中，我們對 Python 語言的過去現在有了一些瞭解，我們準備好了執行 Python 程式所需要的直譯器環境。相信大家已經迫不及待的想開始自己的 Python 程式設計之旅了，但是新問題來了，我們應該在什麼地方書寫 Python 程式，然後又怎麼執行它呢？

### 編寫程式碼的工具

下面我們為大家講解幾種可以編寫和執行 Python 程式碼的工具，大家可以根據自己的需求來選擇合適的工具。當然，對於初學者，我個人比較推薦使用 PyCharm，因為它不需要太多的配置也非常的強大，對新手還是很友好的。如果你也聽說過或者喜歡 PyCharm，可以直接跳過下面對其他工具的介紹，直接快進到講解 PyCharm 的地方。

#### 預設的互動式環境

我們開啟 Windows 的“命令提示符”或“PowerShell”工具，輸入`python`然後按下`Enter`鍵，這個命令會把我們帶到一個互動式環境中。所謂互動式環境，就是我們輸入一行程式碼並按下`Enter`鍵，程式碼馬上會被執行，如果程式碼有產出結果，那麼結果會被顯示在視窗中，如下所示。

```Bash
Python 3.10.10
Type "help", "copyright", "credits" or "license" for more information.
>>> 2 * 3
6
>>> 2 + 3
5
>>>
```

> **說明**：使用 macOS 系統的使用者需要開啟“終端”工具，輸入`python3`進入互動式環境。

如果希望退出互動式環境，可以在互動式環境中輸入`quit()`，如下所示。

```Bash
>>> quit()
```

#### 更好的互動式環境 - IPython

上面說的互動式環境使用者體驗並不怎麼好，大家使用一下就能感受到。我們可以用 IPython 來替換掉它，因為 IPython 提供了更為強大的編輯和互動功能。我們可以在命令提示符或終端中使用 Python 的包管理工具`pip`來安裝 IPython，如下所示。

```bash
pip install ipython
```

> **提示**：在使用上面的命令安裝 IPython 之前，可以先透過`pip config set global.index-url https://pypi.doubanio.com/simple`命令或`pip config set global.index-url https://pypi.tuna.tsinghua.edu.cn/simple/`將下載源修改為國內的豆瓣映象或清華映象，否則下載安裝的過程可能會非常的緩慢。

接下來可以使用下面的命令啟動 IPython，進入互動式環境。

```bash
ipython
```

> **說明**：還有一個網頁版的 IPython 名叫 Jupyter，我們在用得著它的地方再為大家介紹。

### 你好世界

按照行業慣例，我們學習任何一門程式語言寫的第一個程式都是輸出`hello, world`，因為這段程式碼是偉大的丹尼斯·裡奇（C 語言之父，和肯·湯普森一起開發了 Unix 作業系統）和布萊恩·柯尼漢（awk 語言的發明者）在他們的不朽著作《*The C Programming Language*》中寫的第一段程式碼，下面是對應的 Python 語言的版本。

```python
print('hello, world')
```

> **注意**：上面程式碼中的圓括號、單引號都是在英文輸入法狀態下輸入的，如果不小心寫成了中文的圓括號或單引號，執行程式碼時會出現`SyntaxError: invalid character '（' (U+FF08)`或`SyntaxError: invalid character '‘' (U+2018)`這樣的錯誤提示。

上面的程式碼只有一個語句，在這個語句中，我們用到了一個名為`print`的函式，它可以幫助我們輸出指定的內容；`print`函式圓括號中的`'hello, world'`是一個字串，它代表了一段文字內容；在 Python 語言中，我們可以用單引號或雙引號來表示一個字串。不同於 C、C++ 或 Java 這樣的程式語言，Python 程式碼中的語句不需要用分號來表示結束，也就是說，如果我們想再寫一條語句，只需要回車換行即可，程式碼如下所示。此外，Python 程式碼也不需要透過編寫名為`main`的入口函式來使其執行，提供入口函式是編寫可執行的 C、C++ 或 Java 程式碼必須要做的事情，這一點很多程式設計師都不陌生，但是在 Python 語言中它並不是必要的。

```python
print('hello, world')
print('goodbye, world')
```

如果不使用 PyCharm 這樣的整合開發環境，我們也可以直接呼叫 Python 直譯器來執行 Python 程式。我們可以將上面的程式碼儲存成一個名為`example01.py`的檔案，對於Windows 系統，我們假設該檔案在`C:\code`目錄下，我們開啟“命令提示符”或“PowerShell”並輸入下面的命令就可以執行它。

```powershell
python C:\code\example01.py
```

對於 macOS 系統，假設我們的檔案在`/Users/Hao`目錄下，那麼可以在終端中輸入下面的命令來執行程式。

```Bash
python3 /Users/Hao/example01.py
```

> **提示**：如果路徑比較長，不願意手動輸入，我們可以透過拖拽的方式將檔案直接拖到“命令提示符”或“終端”中，這樣會自動輸入完整的檔案路徑。

大家可以試著修改上面的程式碼，比如將單引號中的`hello, world`換成其他內容或者多寫幾個這樣的語句，看看會執行出怎樣的結果。需要提醒大家的是，寫 Python 程式碼時，最好每一行只寫一條語句。雖然，我們可以使用`;`作為分隔將多個語句寫在一行中，但是這樣做會讓程式碼變得非常難看，不再具備良好的可讀性。

### 註釋你的程式碼

註釋是程式語言的一個重要組成部分，用於在程式碼中解釋程式碼的作用，從而達到增強程式碼可讀性的目標。當然，我們也可以將程式碼中暫時不需要執行的程式碼段透過新增註釋來去掉，這樣當你需要重新使用這些程式碼的時候，去掉註釋符號就可以了。簡單的說，**註釋會讓程式碼更容易看懂但不會影響程式碼的執行結果**。

Python 中有兩種形式的註釋：

1. 單行註釋：以`#`和空格開頭，可以註釋掉從`#`開始後面一整行的內容。
2. 多行註釋：三個引號（通常用雙引號）開頭，三個引號結尾，通常用於新增多行說明性內容。

```python
"""
第一個Python程式 - hello, world

Version: 1.0
Author: KevinLuo
"""
# print('hello, world')
print("你好，世界！")
```
