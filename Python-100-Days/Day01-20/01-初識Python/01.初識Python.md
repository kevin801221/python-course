##  初識Python

### Python簡介

Python（英式發音：/ˈpaɪθən/；美式發音：/ˈpaɪθɑːn/）是由荷蘭人吉多·範羅蘇姆（Guido von Rossum）發明的一種程式語言，是目前世界上最受歡迎和擁有最多使用者的程式語言。Python 強調程式碼的可讀性和語法的簡潔性，相較於 C、C++、Java 這些同樣影響深遠的程式語言，Python 讓使用者能夠用更少的程式碼表達自己的意圖。下面是幾個權威的程式語言排行榜給出的 Python 語言的排名，其中第1張圖由 TIOBE Index 提供，第3張圖由 IEEE Spectrum 提供。值得一提的是第2張圖，它展示了程式語言在全球最大程式碼託管平臺 GitHub 上受歡迎的程度，最近的四年時間 Python 語言都佔據了冠軍的寶座。

#### Python編年史

下面是 Python 語言發展過程中的一些重要時間點：

1. 1989年12月：吉多·範羅蘇姆決心開發一個新的指令碼語言及其直譯器來打發無聊的聖誕節，新語言將作為 ABC 語言的繼承者，主要用來替代 Unix shell 和 C 語言實現系統管理。由於吉多本人是 BBC 電視劇《*Monty Python's Flying Circus*》的忠實粉絲，所以他選擇了 Python 這個詞作為新語言的名字。
2. 1991年02月：吉多·範羅蘇姆在 alt.sources 新聞組上釋出了 Python 直譯器的最初程式碼，標記為版本0.9.0。
3. 1994年01月：Python 1.0釋出，夢開始的地方。
4. 2000年10月：Python 2.0釋出，Python 的整個開發過程更加透明，生態圈開始慢慢形成。
5. 2008年12月：Python 3.0釋出，引入了諸多現代程式語言的新特性，但並不完全向下相容。
6. 2011年04月：pip 首次釋出，Python 語言有了自己的包管理工具。
7. 2018年07月：吉多·範羅蘇姆宣佈從“終身仁慈獨裁者”（開源專案社群出現爭議時擁有最終決定權的人）的職位上“永久休假”。
8. 2020年01月：在 Python 2和 Python 3共存了11年之後，官方停止了對 Python 2的更新和維護，希望使用者儘快切換到 Python 3。
9. 目前：Python 在大模型（GPT-3、GPT-4、BERT等）、計算機視覺（影象識別、目標檢測、影象生成等）、智慧推薦（YouTube、Netflix、位元組跳動等）、自動駕駛（Waymo、Apollo等）、語音識別、資料科學、量化交易、自動化測試、自動化運維等領域都得到了廣泛的應用，Python 語言的生態圈也是相當繁榮。

> **說明**：大多數軟體的版本號一般分為三段，形如A.B.C，其中A表示大版本號，當軟體整體重寫升級或出現不向後相容的改變時，才會增加A；B表示功能更新，出現新功能時增加B；C表示小的改動（例如：修復了某個Bug），只要有修改就增加C。

#### Python優缺點

Python 語言的優點很多，簡單為大家列出幾點。

1. **簡單優雅**，跟其他很多程式語言相比，Python **更容易上手**。
2. 能用更少的程式碼做更多的事情，**提升開發效率**。
3. 開放原始碼，擁有**強大的社群和生態圈**。
4. **能夠做的事情非常多**，有極強的適應性。
5. **膠水語言**，能夠黏合其他語言開發的東西。
6. 解釋型語言，更容易**跨平臺**，能夠在多種作業系統上執行。

Python 最主要的缺點是**執行效率低**（解釋型語言的通病），如果更看重程式碼的執行效率，C、C++ 或 Go 可能是你更好的選擇。

### 安裝Python環境

工欲善其事，必先利其器。想要開始你的 Python 程式設計之旅，首先得在計算機上安裝 Python 環境，簡單的說就是安裝執行 Python 程式需要的 Python 直譯器。我們推薦大家安裝官方的 Python 3 直譯器，它是用 C 語言編寫的，我們通常也稱之為 CPython，它可能是你目前最好的選擇。首先，我們需要從官方網站的[下載頁面](https://www.python.org/downloads/)找到下載連結，點選“Download”按鈕進入下載頁面後，需要根據自己的作業系統選擇合適的 Python 3安裝程式，如下圖所示。

<img src="../res/day01/python_download_page_1.png" style="zoom:40%;">

進入下載頁面後，有些 Python 版本並沒有提供 Windows 和 macOS 系統的安裝程式，只提供了原始碼的壓縮檔案，對於熟悉 Linux 系統的小夥伴，我們可以透過原始碼構建安裝；對於使用 Windows 或 macOS 系統的小夥伴，我們還是**強烈建議**使用安裝程式。例如，你想安裝 Python 3.10，選擇 Python 3.10.10 或 Python 3.10.11 就能找到 Windows 或 macOS 系統的安裝包，而其他版本可能只有原始碼，如下圖所示。

#### Windows環境

下面我們以 Windows 11為例，講解如何在 Windows 作業系統上安裝 Python 環境。雙擊執行從官網下載的安裝程式，會開啟一個安裝嚮導，如下圖所示。

<img src="../res/day01/install_python_1.png" style="zoom:50%;">

首先，一定要記得勾選“Add python.exe to PATH”選項，它會幫助我們將 Python 直譯器新增到 Windows 系統的 PATH 環境變數中（不理解沒關係，勾上就對了）；其次，“Use admin privileges when installing py.exe”是為了在安裝過程中獲得管理員許可權，建議勾選。然後，我們選擇“Customize Installation”，使用自定義安裝的模式，這是專業人士的選擇，而你就（假裝）是那個專業人士，不建議使用“Install Now”（預設安裝）。

接下來，安裝嚮導會提示你勾選需要的“Optional Features”（可選特性），這裡咱們可以直接全選。值得一提的是其中的第2項，它是 Python 的包管理工具 pip，可以幫助我們安裝三方庫和三方工具，所以一定要記得勾選它，然後點選“Next”進入下一環節。

<img src="../res/day01/install_python_2.png" style="zoom:50%;">

接下來是對“Advanced Options”（高階選項）的選擇，這裡我們建議大家只勾選“Add Python to environment variables”和“Precompile standard library”這兩個選項，前者會幫助我們自動配置好環境變數，後者會預編譯標準庫（生成`.pyc`檔案），這樣在使用時就無需臨時編譯了。還是那句話，不理解沒關係，勾上就對了。下面的“Customize install location”（自定義安裝路徑）**強烈建議**修改為自定義的路徑，這個路徑中不應該包含中文、空格或其他特殊字元，注意這一點會為你將來減少很多不必要的麻煩。設定完成後，點選“Install”開始安裝。

<img src="../res/day01/install_python_3.png" style="zoom:50%;">

安裝成功會出現如下圖所示的畫面，安裝成功的關鍵詞是“successful”，如果安裝失敗，這裡的單詞會變成“failed”。

<img src="../res/day01/install_python_4.png" style="zoom:50%;">

安裝完成後可以開啟 Windows 的“命令列提示符”或 PowerShell，然後輸入`python --version`或`python -V`來檢查安裝是否成功，這個命令是檢視 Python 直譯器的版本號。如果看到如下所示的畫面，那麼恭喜你，Python 環境已經安裝成功了。這裡我們建議再檢查一下 Python 的包管理工具 pip 是否可用，對應的命令是`pip --version`或`pip -V`。

#### 其他安裝方式

有人可能會推薦新手直接安裝 [Anaconda](https://www.anaconda.com/download/success)，因為 Anaconda 會幫助我們安裝 Python 直譯器以及一些常用的三方庫，除此之外還提供了一些便捷的工具，特別適合萌新小白。我個人並不推薦這種方式，因為在安裝 Anaconda 時你會莫名其妙安裝了一大堆有用沒用的三方庫（佔用比較多的硬碟空間），然後你的終端或命令提示符會被 Anaconda 篡改（每次啟動自動啟用虛擬環境），這些並不符合軟體設計的**最小驚訝原則**。其他關於 Anaconda 的小毛病此處就不再贅述了，如果你非要使用 Anaconda，推薦安裝 Miniconda，它跟 Anaconda 在同一個下載頁面。

還有萌新小白經常會聽到或說出，“我要寫 Python 程式，安裝一個 PyCharm 不就可以了嗎？”。這裡簡單科普一下，PyCharm 只是一個輔助寫 Python 程式碼的工具，它本身並不具備執行 Python 程式碼的能力，執行 Python 程式碼靠的是我們上面安裝的 Python 直譯器。當然，有些 PyCharm 版本在建立 Python 專案時，如果檢測不到你電腦上的 Python 環境，也會提示你聯網下載 Python 直譯器。PyCharm 的安裝和使用我們放在了下一課。

### 總結

總結一下我們學到的東西：

1. Python 語言很強大，可以做很多的事情，所以值得我們去學習。
2. 要使用 Python語言，首先得安裝 Python 環境，也就是執行 Python 程式所需的 Python 直譯器。
3. Windows 系統可以在命令提示符或 PowerShell 中輸入`python --version`檢查 Python 環境是否安裝成功；macOS 系統可以在終端中輸入`python3 --version`進行檢查。
