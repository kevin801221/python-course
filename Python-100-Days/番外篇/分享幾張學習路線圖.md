## 分享幾張學習路線圖

今天逛了一個名為 [Developer Roadmaps](https://roadmap.sh) 的網站，上面就有好多的學習路線圖。其中有一張關於 Python 的學習路線圖還不錯，分享給大家。如果用 Python 做工程化開發（後端開發、介面開發、應用開發等），這張圖上的內容基本都是需要涉獵的；如果僅僅是將 Python 作為輔助工具，學習之前給大家整理的[《從零開始學Python》](https://www.zhihu.com/column/c_1216656665569013760)的 20 節課就已經足夠了。近期，我也在重置這 20 節課的內容，重置後的內容會標記為“2025版”，也希望在這個過程中聽到大家的意見和建議。

![](res/Python-Roadmap.png)

這裡還有三個相關的學習路線圖，分別是後端開發（Backend）、開發運維（DevOps）和資料科學（Data Science），後端開發的學習路線圖如下所示，內容比較龐雜，核心的程式語言是 Java 或 Go。我個人不推薦大家用 Python 做後端開發，如果 Python 語言已經掌握得很好了，同時也具備了較多的電腦科學相關知識，可以嘗試 AI 應用開發這個賽道，我相信你會有更多的收穫。大家可以注意下用紫色勾選的專案，這些是作者推薦的選項，大部分也是我想推薦給後端開發者學習的內容。

![](res/Backend-Roadmap.png)

我們再看看開發運維的學習路線圖，目前 DevOps 也是招聘市場的熱門崗位，有興趣的小夥伴可以關注下。

![](res/DevOps-Roadmap.png)

如果學過一些 Python 相關知識，想以此為基礎轉行做資料分析，可以看看下面這個學習路線圖。需要說明的是，Python 僅僅是做資料分析的一個工具，要成為一個優秀的資料分析師需要有良好的資料思維，同時還要積累足夠的業務知識，這樣才能從資料中產生商業洞察，用資料最佳化產品、驅動業務，真正為決策提供強有力的支撐。

![](res/Data-Analyst-Roadmap.png)
